SVD & PCA

Low-rank spatial models versus a color-space rotation ahead of the DCT.

Singular value decomposition (SVD)

SVD expresses a 2D channel as U Σ Vᵀ. Truncating small singular values yields a controlled rank approximation: fewer parameters to store, smooth denoising-like behavior, and a direct knob (how many singular vectors to keep) on rate versus distortion.

We explored SVD as an alternative spatial representation to DCT blocks. It is flexible but can be expensive on large matrices and needs careful quantization of U, Σ, and Vᵀ factors for a practical codec.

PCA color prefilter

In the hybrid Huffman pipeline we run PCA across the three RGB channels at each pixel (treating the whole image as a set of 3-vectors). That produces decorrelated “eigen-channels” so energy in chroma can be quantized more aggressively than the first principal component, which behaves like a luminance-heavy axis.

The rotation matrix Vᵀ and per-channel means are stored as metadata so the decoder can invert PCA after inverse DCT and deblocking. See codec/encoders/pca.py and the hybrid compressor for the exact pre/post steps.
